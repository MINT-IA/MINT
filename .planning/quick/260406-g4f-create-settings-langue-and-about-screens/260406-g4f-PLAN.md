1. Add imports at the top of app.dart:

2. Add two GoRoute entries just BEFORE the `// -- OUTILS & DIVERS` comment block (around line 867). Insert them as top-level routes (not nested under /profile), with `parentNavigatorKey: _rootNavigatorKey`:

   ```dart
   // -- SETTINGS ---------------------------------------------------
   GoRoute(
     path: '/settings/langue',
     parentNavigatorKey: _rootNavigatorKey,

   // -- ABOUT ------------------------------------------------------

     path: '/about',
     parentNavigatorKey: _rootNavigatorKey,
     builder: (context, state) => const AboutScreen(),
   ),
   ```

**2. Create `apps/mobile/lib/providers/tension_card_provider.dart`:**

`TensionCardProvider extends ChangeNotifier` with:
- `List<TensionCard> _cards = []` (always exactly 3 when data exists, empty when no data)
- `CleoLoopPosition _loopPosition = CleoLoopPosition.insight` (default)
- `bool _isLoading = true`
- `bool get isEmpty => _cards.isEmpty`
- `bool get isLoading => _isLoading`

`Future<void> refresh()` method:
1. Call `CommitmentService().getCommitments()` wrapped in try/catch (return empty on error)
2. Call `FreshStartService().fetchLandmarks()` wrapped in try/catch (return empty on error)
3. Read conversation count from `ConversationStore` (use SharedPreferences directly to read the index key — `ConversationStore` uses `_chat_conversation_index` prefixed by user ID)
4. Load `PartnerEstimateService().load()` wrapped in try/catch

Build exactly 3 cards using `_selectTensions()`:
- **Earned card (past)**: Pick from completed commitments (status == 'completed'), uploaded documents, or partner declared. Priority: most recent completed commitment. Fallback: "Premiere conversation" if any conversation exists. If nothing, this slot is null.
- **Pulsing card (present)**: Pick from active commitments (status == 'active'), open conversations, or pending couple questions. Priority: most recent active commitment. Fallback: active conversation count > 0.
- **Ghosted card (future)**: Pick from upcoming fresh-start landmarks (nearest by daysUntil), or retirement projection gap. Priority: nearest landmark. Fallback: generic "Ton avenir financier".

If ALL three slots are null, set `_cards = []` (triggers empty state in UI).
If at least one slot has data, fill missing slots with sensible defaults:
- Missing earned: use "Premiere conversation" if conversations > 0
- Missing pulsing: use "Parle au coach" prompt
- Missing ghosted: use "Ton avenir financier" generic

Determine `_loopPosition` from most recent user activity:
- Last action was document upload or commitment accept -> `CleoLoopPosition.action`
- Last action was coach conversation -> `CleoLoopPosition.conversation`
- Fresh-start message received -> `CleoLoopPosition.insight`
- Default -> `CleoLoopPosition.insight`

Call `notifyListeners()` after building cards.

**Deep links for card taps:**
- Earned -> `/coach/chat?prompt=Montre-moi+mes+engagements+tenus` (commitment history via coach)
- Pulsing -> `/coach/chat` (coach chat to continue tension)
- Ghosted -> `/coach/chat?prompt=Parle-moi+de+mon+prochain+jalon` (projection via coach)

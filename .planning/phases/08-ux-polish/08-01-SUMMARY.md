## Decisions Made

- `ConfidenceScoreCard` is a pure StatelessWidget — score computed inline in MintHomeScreen Builder via `ConfidenceScorer.scoreEnhanced(profile)`. No separate ConfidenceScoreProvider created because the scorer is a pure local function with no async operations, matching the threat model's T-08-02 mitigation (called only on profile change via watch, not every frame).
- `disableAnimations` guard set to `Duration.zero` on `AnimatedSwitcher` duration/reverseDuration — cleaner than conditional widget tree rebuild, preserves FadeTransition in tree.
- Added `super.key` to `_SignalProactifCard` and `_RadarAnticipateCard` constructors (previously missing). This was required for `AnimatedSwitcher` to detect ValueKey-based child swaps — without it, Dart emits an error.

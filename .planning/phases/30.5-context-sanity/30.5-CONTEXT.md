Implémentable J1-J2 sans dépendance lints Phase 34 (early-version of `accent_lint_fr.py` ship en CTX-02 si nécessaire).

  - **(c) token cost per session** = parse `.claude/projects/-Users-julienbattaglia-Desktop-MINT/*.jsonl` transcripts (déjà sur disque, 0 API call Anthropic) → somme `usage.input_tokens + usage.output_tokens` par session.
  - **(d) time-to-first-correct-output** = harness offline 20 golden prompts (run sur `claude --headless` ou équivalent) → mesure # tours avant 1er output validant accents/banned terms/i18n via les mêmes lints que (a). Run nightly ou on-demand.

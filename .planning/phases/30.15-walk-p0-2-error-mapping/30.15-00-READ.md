## Changes

### Service layer (`coach_chat_api_service.dart`)
Added `errorType` field to fallback payload, differentiating:
- `'network'` — `ClientException` or `TimeoutException`
- `'service'` — HTTP 503 or any 5xx (backend unavailable, typical when
  `ANTHROPIC_API_KEY` missing or Railway 5xx)
- `'session'` — HTTP 400 (invalid anonymous session UUID)
- `'unknown'` — ultimate fallback (any other failure)

Preserves existing 429 rate-limit behavior (returns detail message).

### UI layer (`anonymous_chat_screen.dart`)
`switch (errorType)` maps to appropriate ARB key via Dart 3 pattern:
```dart
final text = switch (errorType) {
  'network' => l.anonymousChatErrorNetwork,
  'service' => l.anonymousChatErrorService,
  'session' => l.anonymousChatErrorSession,
  _ => l.anonymousChatError,
};
```

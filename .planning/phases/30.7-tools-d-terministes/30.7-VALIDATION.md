- **After every task commit:** Run `pytest tests/ -q`
- **After every plan wave:** Run full suite + `claude_md_bracket.py` dry-run
- **Before `/gsd-verify-work`:** Full suite must be green + 3-session J0 gate
- **Max feedback latency:** 45 seconds

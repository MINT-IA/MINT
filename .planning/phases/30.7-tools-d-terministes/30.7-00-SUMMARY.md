## Decisions Made

- **Pin ceiling `mcp<2.0`** — major breaking changes documented in Pitfall 10 (FastMCP decorator renames). Floor `>=1.9` because mcp<1.9 drops `FastMCP` stdio helper per upstream changelog.
- **Extract scan_text inline rather than create a new module** — Phase 34 lefthook already greps `tools/checks/accent_lint_fr.py` for `def scan_file`. Adding a sibling file would fragment discovery.
- **Ship pytest-asyncio in Wave 0** — Wave 2 needs it for MCP stdio integration tests. Installing it now avoids a second `pip install` commit 3 days from now.
- **Baseline JSON committed, not gitignored** — Wave 3 `--assert-delta 30` mode needs the baseline checked into the tree so verifier agents can reproduce from a clean checkout.
- **README uses technical English only** — `tools/mcp/mint-tools/` is a dev-time sub-project, never user-facing. French i18n + banned-terms scan would be noise.

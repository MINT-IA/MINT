Build the frontend anonymous chat experience: session service, chat screen, intent-to-chat routing, and auth gate conversion UX.

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-13-07 | Tampering | Intent query parameter | mitigate | URI-encode intent in navigation, Pydantic validates on backend (max 2000 chars) |
| T-13-08 | Spoofing | Auth gate bypass | mitigate | `_isAuthGateLocked` state prevents message sending after 3rd message; backend enforces 429 as ultimate guard |
| T-13-09 | Information Disclosure | Anonymous messages in SharedPreferences | accept | Messages are on user's own device only; PII scrubbing applied before storage; acceptable per user decision (frontend-only persistence) |
| T-13-10 | Denial of Service | Rapid message sending | mitigate | Backend rate limit (3 lifetime) is the hard stop; frontend disables send button during loading |
</threat_model>

- Atomic commits per layer is standard MINT git discipline (CLAUDE.md ¬ß4).
- Backend = source of truth for constants and formulas; Flutter mirrors. The rename order respects this: backend renames first (B1-B4), then mobile mirrors (F1-F5).

### Integration Points
- F1 (model layer rename) propagates through F4 (services consumers) to F5 (widgets) to F2/F3 (screens).

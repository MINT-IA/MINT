**§13 — Anti-examples appendix (20 phrases that look right and are wrong)**

Intro paragraph: these are NOT in the frozen corpus (`tools/voice_corpus/frozen_phrases_v1.json`). They are the perceptual inverse — phrases the Krippendorff raters will NOT see during classification but that Phase 11 ComplianceGuard regression (VOICE-08) will test against. Each anti-example is tagged with the specific anti-shame checkpoint(s) violated, the N level it superficially mimics, the failure family, a one-line "why this fails" gloss, and a corrected form.

**6 failure families** (distribute the 20 anti-examples across them roughly evenly):

1. **Prescription drift** (~4 examples): phrases that use imperative without conditional softening. Violates checkpoint #3. Typical look-right pattern: "Tu devrais faire X" without "pourrais / envisager / peut-être".
2. **Comparison** (~3 examples): phrases comparing the user to other Swiss users, age cohorts, "people in your situation". Violates checkpoint #1. Banned by CLAUDE.md §6 "No-Social-Comparison" rule. The subtle version is the hardest to catch: "80% des gens dans ta situation…" or "La plupart des indépendants à ton âge…".
3. **Shame induction** (~4 examples): phrases that imply the user is late, behind, ignorant, or should have done something already. Violates checkpoint #6 (and often #1). The subtle version: "Il est encore temps de commencer" — the word "encore" silently asserts the user is late.
4. **Tone-lock / false intensity** (~3 examples): phrases at N4 or N5 register that are actually just verbose N2 content dressed up with punctuation breaks. The failure is that the intensity comes from formatting, not from meaning. Phase 11 reverse-Krippendorff (VOICE-06) catches this kind of drift; §13 documents the pattern so authors recognize it.
5. **Banned terms at high register** (~3 examples): phrases using "garanti", "sans risque", "optimale", "meilleur", "parfait", "conseiller" (see CLAUDE.md §6). Often hidden inside otherwise-good phrases. Violates compliance and checkpoint #3 (prescription by absolute-framing).
6. **Sensitivity violation** (~3 examples): phrases that would score as N4 or N5 in a neutral context but land on a sensitive topic from v0.5 §5 (deuil, divorce, perteEmploi, maladieGrave, etc.). Violates the hard N3 cap. The subtle version: a N4-feeling phrase "correctly" routed to a user tagged with `perteEmploi` because the author forgot the cap. Phase 11 VOICE-09 server gate catches this at runtime; §13 documents it so authors recognize it at write time.

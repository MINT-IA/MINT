    **AnticipationProvider** — Create `anticipation_provider.dart`:

    ```dart
    class AnticipationProvider extends ChangeNotifier {
      AnticipationRankResult _rankResult = AnticipationRankResult(visible: [], overflow: []);
      bool _evaluated = false;

      List<AnticipationSignal> get visibleSignals => _rankResult.visible;
      List<AnticipationSignal> get overflowSignals => _rankResult.overflow;
      bool get hasSignals => _rankResult.visible.isNotEmpty;
      bool get hasOverflow => _rankResult.overflow.isNotEmpty;
    }
    ```

    Key methods:
    1. `Future<void> evaluateOnSessionStart({required CoachProfile profile, required List<BiographyFact> facts, DateTime? now})`:
       - If already evaluated this session (`_evaluated`), return early (per CTX-02: once per session)
       - Get SharedPreferences instance
       - Get dismissed triggers -> map to signal IDs
       - Get snoozed triggers -> add to dismissed
       - Get signalsShownThisWeek
       - Call `AnticipationEngine.evaluate(profile: profile, facts: facts, now: now ?? DateTime.now(), dismissedIds: allDismissedIds)`
       - Call `AnticipationRanking.rank(signals: raw, now: now ?? DateTime.now(), signalsAlreadyShownThisWeek: shownCount)`
       - For each visible signal: validate text with `ComplianceGuard.validateAlert()` using resolved ARB text. If not compliant, move to overflow (never show non-compliant alert).
       - Record each visible signal as shown (`AnticipationPersistence.recordSignalShown`)
       - Store result, set `_evaluated = true`, `notifyListeners()`

    2. `Future<void> dismissSignal(AnticipationSignal signal)`:
       - Map signal.template to AnticipationTrigger
       - Call `AnticipationPersistence.dismiss(prefs, trigger)`
       - Remove from visible, notifyListeners()

    3. `Future<void> snoozeSignal(AnticipationSignal signal)`:
       - Map signal.template to AnticipationTrigger
       - Call `AnticipationPersistence.snooze(prefs, trigger)`
       - Remove from visible, notifyListeners()

    4. `void resetSession()` — sets `_evaluated = false` for next app launch

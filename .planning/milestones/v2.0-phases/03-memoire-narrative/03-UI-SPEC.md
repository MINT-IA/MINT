# Phase 3 — UI Design Contract

> Visual and interaction contract for Memoire Narrative. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | Flutter Material 3 (existing) |
| Preset | not applicable |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, label-to-value gap |
| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Card internal padding, list item spacing |
| lg | 24px | Screen horizontal padding, section padding |
| xl | 32px | Between major sections on privacy screen |
| 2xl | 48px | Hero section breathing room |
| 3xl | 64px | Page-level top/bottom spacing |

Exceptions: Touch targets 48px minimum on all interactive elements (edit, delete, expand/collapse buttons).

---

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Display | 48px | w700 | 1.1 | Montserrat |
| Heading | 26px | w700 | 1.2 | Montserrat |
| Body | 16px | w400 | 1.5 | Inter |
| Label | 14px | w400 | 1.4 | Inter |

Usage per screen:
- **Privacy screen title**: Heading (26px w700)
- **Fact value** (e.g. "~100k CHF"): Display (48px w700) -- only on summary hero stat if present
- **Fact label** (e.g. "Salaire brut"): Body (16px w400)
- **Source + date metadata** (e.g. "Certificat LPP -- mars 2026"): Label (14px w400)
- **Stale warning text**: Label (14px w400) in MintColors.warning
- **Section headers** (e.g. "Donnees financieres", "Evenements de vie"): Heading (26px w700)

## Screens

### Screen 1: Privacy Control ("Ce que MINT sait de toi")

**Category**: Form Screen (D) -- data management
**Route**: /profile/privacy-control (accessible from ProfileDrawer)
**Background**: MintColors.porcelaine

**Layout (top to bottom)**:
1. **AppBar**: White background, title "Ce que MINT sait de toi" (Heading 26px w700), back arrow
2. **Summary stat** (optional hero): fact count + freshness percentage -- e.g. "14 donnees | 86\u00a0% a jour" (Body 16px w400, centered)
3. **Section list** (grouped by category): each section is a Heading (26px w700) followed by fact cards
4. **Bottom safe area**: 24px padding

**Fact card (MintSurface)**:
- Left column: fact label (Body 16px w400, MintColors.textSecondary) + fact value (Body 16px w400, MintColors.textPrimary)
- Right column: edit icon (48px touch target) + delete icon (48px touch target)
- Bottom row: source badge (Label 14px w400) + date (Label 14px w400, MintColors.textMuted) + freshness indicator
- Stale facts: yellow left border (4px, MintColors.warning) + "Donnees datant de {X} mois" label below date
- Card padding: 16px internal (MintSpacing.md)
- Card spacing: 8px between cards (MintSpacing.sm)

**Sections** (grouped):
- Donnees financieres (salary, LPP, 3a, etc.)
- Evenements de vie (marriage, job change, etc.)
- Decisions (confirmed extractions, user edits)

**Interactions**:
- Tap edit icon -> inline edit bottom sheet (value field + save button)
- Tap delete icon -> destructive confirmation dialog
- Swipe left on card -> reveal delete button (secondary gesture)
- Pull to refresh -> recalculate freshness decay

### Screen 2: Fact Edit Bottom Sheet

**Category**: Form Screen (D)
**Trigger**: Tap edit icon on any fact card

**Layout**:
1. **Handle bar**: centered, 32px wide, MintColors.border
2. **Title**: fact label (Heading 26px w700) -- e.g. "Salaire brut"
3. **Current value display**: current value + source + date (Body 16px w400)
4. **Input field**: MintAmountField or MintTextField (existing premium widgets), pre-filled with current value
5. **Source note**: "Source\u00a0: modification manuelle" (Label 14px w400, MintColors.textMuted)
6. **CTA**: "Enregistrer" button (48px height, full width, MintColors.primary background, white text)
7. **Bottom padding**: 24px + safe area

### Screen 3: Delete Confirmation Dialog

**Category**: Overlay
**Trigger**: Tap delete icon or swipe-to-delete

**Layout**:
- Title: "Supprimer cette donnee\u00a0?" (Heading 26px w700)
- Body: "Cette action est irreversible. MINT oubliera {fact_label}." (Body 16px w400)
- Cancel: "Annuler" (text button, MintColors.textSecondary, 48px touch target)
- Confirm: "Supprimer" (text button, MintColors.error, 48px touch target)

## Coach UI Integration (No New Screen)

The coach integration for BIO-03/BIO-04/BIO-07/BIO-08 does not require new screens. It modifies the existing coach system prompt injection via `context_injector_service.dart`. Visual impacts:

- **Coach bubble referencing biography**: No visual change -- coach text already renders in MintColors.coachBubble (#FCFBF8) with MintColors.coachAccent (#3A3D44) text
- **Refresh prompt from coach** (BIO-08): Appears as a regular coach message with a contextual suggestion chip below -- "Rescanner mon certificat" chip (MintColors.accentPastel background, MintColors.textPrimary text, 48px touch target height, 8px border radius)
- **Dated references** (COMP-02): Coach text includes parenthetical dates naturally -- e.g. "(selon ton certificat de mars 2025)" -- no special visual treatment beyond standard body text

## Freshness Visual System

| Freshness State | Age | Visual Treatment |
|-----------------|-----|------------------|
| Fresh | 0-6 months | MintColors.success dot (8px) + "A jour" label |
| Aging | 6-12 months (annual) / 2-3 months (volatile) | MintColors.warning dot (8px) + "Donnees datant de {X} mois" |
| Stale | > 12 months (annual) / > 3 months (volatile) | MintColors.error dot (8px) + yellow card border (4px left) + "Donnees obsoletes" + excluded badge |

The dot is an 8px circle positioned 4px left of the date label. The left border on stale cards is 4px wide, MintColors.warning color.

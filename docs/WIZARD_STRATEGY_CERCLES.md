# Stratégie Wizard MINT : Théorie des Cercles

## Philosophie : L'Approche par Cercles Concentriques

En finance suisse, la planification suit une logique de **cercles concentriques** :
Chaque cercle doit être **solide** avant de passer au suivant.

```
┌─────────────────────────────────────┐
│  CERCLE 4 : OPTIMISATION            │  (Immobilier, Succession, Offshore)
│  ┌───────────────────────────────┐  │
│  │ CERCLE 3 : CROISSANCE         │  │  (3a Actions, Titres, Crypto)
│  │  ┌─────────────────────────┐  │  │
│  │  │ CERCLE 2 : PRÉVOYANCE   │  │  │  (3a, Rachat LPP, AVS volontaire)
│  │  │  ┌───────────────────┐  │  │  │
│  │  │  │ CERCLE 1 : BASE   │  │  │  │  (Fonds urgence, Dettes, LAMal)
│  │  │  └───────────────────┘  │  │  │
│  │  └─────────────────────────┘  │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

## CERCLE 1 : PROTECTION & SÉCURITÉ (Priorité Absolue)

### Objectif
**Ne jamais être en détresse financière immédiate.**

### Questions Clés
1. **Fonds d'urgence**
   - Question : "As-tu 3-6 mois de charges de côté ?"
   - Réponse : Si NON → Bloquer Cercle 2 et recommander épargne automatique
   
2. **Dettes toxiques**
   - Question : "As-tu des dettes de consommation (crédit, leasing) ?"
   - Réponse : Si OUI → Recommander remboursement avant 3a/investissements
   - **Pourquoi ?** Un crédit à 8% coûte plus que ce qu'un 3a rapporte (2-3%)

3. **Assurances obligatoires**
   - LAMal : Vérifier prime/franchise optimale
   - RC Ménage : Vérifier si souscrite (obligation légale de fait)
   - **Si indépendant** : Perte de gain obligatoire

### Recommandations Cercle 1
- ❌ **NE PAS** ouvrir un 3a si tu as 5000 CHF de crédit conso à 9%
- ✅ **PRIORITÉ** : Rembourser dettes > Fonds urgence > Assurances base

## CERCLE 2 : PRÉVOYANCE FISCALE (Une fois la base sécurisée)

### Objectif
**Maximiser la déduction fiscale tout en gardant la flexibilité.**

### 3a : LES CHOIX CRUCIAUX

#### Question : "Combien de comptes 3a as-tu ?"
**Réponse optimale : 2-3 comptes minimum**

**Pourquoi plusieurs comptes ?**
- **Optimisation fiscale au retrait** : Chaque retrait 3a est imposé séparément et progressivement
- **Exemple concret** :
  - 1 compte de 100'000 CHF → Impôt ~13'000 CHF (taux moyen 13%)
  - 2 comptes de 50'000 CHF retirés sur 2 ans → Impôt ~5'000 x 2 = 10'000 CHF
  - **Économie : 3'000 CHF** juste en ayant 2 comptes au lieu d'un !

#### Question : "Où est ton 3a actuellement ?"
**Options avec éducation intégrée :**

##### Option A : 3a Bancaire/Assurance Classique (ex: UBS, Crédit Suisse, AXA)
- **Rendement** : 0.5% - 1.5%/an
- **Frais** : 0.5% - 2%/an
- **Flexibilité** : Faible (assurance = verrouillé jusqu'à 60/65 ans)
- ⚠️ **Piège** : Assurance-vie 3a = commissions cachées énormes (jusqu'à 5% la 1ère année)

##### Option B : 3a Titres Nouvelle Génération (VIAC, Finpension, frankly)
- **Rendement historique** : 4% - 7%/an (selon stratégie)
- **Frais** : 0.39% - 0.68%/an (jusqu'à **5x moins cher** qu'une banque)
- **Flexibilité** : Totale (change de stratégie en 1 clic)
- **Fournisseurs recommandés** :
  - **VIAC** : Leader, stratégies 0-100% actions, app fluide
  - **Finpension** : Le moins cher (0.39%), idéal gros capitaux
  - **frankly (ZKB)** : Sécurité banque cantonale, frais corrects

##### Option C : 3a Assurance Risque Pur (RARE)
**Quand choisir cette option ?**
- ❌ **JAMAIS comme épargne** (rendement catastrophique)
- ✅ **UNIQUEMENT si besoin couverture décès** ET que :
  - Tu as des personnes à charge (enfants, conjoint sans revenu)
  - Tu n'as pas de LPP (indépendant sans 2e pilier)
  - Montant couverture : 3-5x ton salaire annuel

**Alternative recommandée** : 3a VIAC/Finpension + Assurance-vie Temporaire séparée (beaucoup moins cher)

## CERCLE 3 : CROISSANCE (Une fois Cercle 1+2 maîtrisés)

### Rachat LPP : L'Arme Fiscale des 45-60 ans

#### Question : "Peux-tu racheter ta LPP ?"
**Sous-questions :**
- "Montant rachetable total" (demander certificat LPP)
- "Projet dans les 3 ans ?" (Si achat immo/retrait anticipé → rachat interdit 3 ans)

#### Stratégie Rachat Échelonné
**Exemple concret (ton profil) :**
- Age : 49 ans
- Rachat possible : 200'000 CHF
- Revenu imposable ménage : 93'600 CHF (7'800 x 12)

**Stratégie optimale :**
1. **Année 1 (2026)** : Rachat 50'000 CHF
   - Économie fiscale : ~15'000 CHF (taux marginal ~30% en VS)
2. **Année 2 (2027)** : Rachat 50'000 CHF
   - Économie : ~15'000 CHF
3. **Année 3 (2028)** : Rachat 50'000 CHF
   - Économie : ~15'000 CHF
4. **Année 4 (2029)** : Rachat 50'000 CHF
   - Économie : ~15'000 CHF

**Total économisé : ~60'000 CHF d'impôts** sur 200'000 investis !

⚠️ **Piège à éviter** : Racheter 200k d'un coup = économie moindre (plafonnement déduction)

## CERCLE 4 : OPTIMISATION & TRANSMISSION

### Immobilier
- Fonds propres 20% disponibles ?
- Capacité hypothécaire (revenu > 3x charges) ?

### Hors-Pilier (Actions, ETF, Crypto)
- Uniquement si Cercles 1-3 remplis
- Horizon 10+ ans
- Max 10-20% du patrimoine

## Navigation Wizard Révisée (Théorie des Cercles)

### Phase 1 : CONTEXTE (Questions 1-6)
1. Prénom (optionnel, personnalisation)
2. Année naissance (calcul horizon retraite)
3. Canton (fiscalité)
4. Statut civil (imposition, déductions)
5. Enfants à charge (déductions familiales)
6. Statut professionnel (LPP oui/non, indépendant = stratégies différentes)

### Phase 2 : CERCLE 1 - DIAGNOSTIC PROTECTION (Questions 7-12)
7. "As-tu 3-6 mois de charges en épargne liquide ?"
   - Si NON → **Alerte Critique** : "Avant d'investir, sécurise d'abord..."
8. "As-tu des dettes de consommation ?" (crédit, leasing hors hypothèque)
   - Si OUI → **Bloquer 3a** : "Rembourse d'abord, tu économiseras plus"
9. Revenu net mensuel ménage
10. Loyer/Hypothèque
11. Charges fixes (assurances, abonnements)
12. LAMal : Franchise actuelle ?

**→ Calcul Clarté Cercle 1 : Si < 80% → Recommandations Protection avant Prévoyance**

### Phase 3 : CERCLE 2 - PRÉVOYANCE FISCALE (Questions 13-20)
13. "Combien de comptes 3a as-tu ?" (0 / 1 / 2 / 3+)
    - Si 0 → Recommander ouverture chez VIAC/Finpension
    - Si 1 → **Alerte éducative** : "Ouvre un 2e pour optimiser le retrait futur"
    - Si 3+ → GG, tu maîtrises !

14. "Où sont tes 3a actuellement ?" (Multi-select)
    - [ ] Banque classique (UBS, CS, Raiffeisen...)
    - [ ] Assurance (AXA, Zurich, SwissLife...)
    - [ ] VIAC
    - [ ] Finpension
    - [ ] frankly
    - [ ] Autre

15. **Insert didactique** : Comparaison rendements VIAC vs Banque sur ton horizon

16. "Verses-tu le maximum 3a chaque année ?" (7'258 CHF si salarié)
    - Si NON → Calculer économie fiscale perdue

17. "Es-tu affilié LPP ?" (Si salarié = oui normalement)

18. **Si 45+ ans** : "Peux-tu racheter ta LPP ?"
    - Montant rachetable (slider 0-300k)
    - **Insert** : Stratégie rachat échelonné avec calcul fiscal

19. "As-tu un projet immobilier dans les 3 ans ?"
    - Si OUI + rachat LPP envisagé → **Alerte légale** : Incompatible !

20. AVS : Lacunes de cotisation ? (Extrait compte AVS)

### Phase 4 : CERCLE 3 - CROISSANCE (Questions 21-25)
21. "As-tu des investissements hors-pilier ?" (Actions, ETF, Crypto)
22. Patrimoine immobilier ?
23. Objectifs long terme (retraite anticipée, achat, héritage)

### Phase 5 : RAPPORT FINAL
- **Scoreboard par cercle** (Cercle 1: 95% ✅ | Cercle 2: 60% ⚠️ | ...)
- **Actions prioritaires** (ex: "Ouvre 2e compte 3a chez VIAC")
- **Estimation fiscale annuelle**
- **Projection retraite** (capital estimé à 65 ans)
